# Rit 
## Rit is a linux command to rename photos to the time as they were taken .
### Getting Started

### Install 

first you must clone :
```
git clone https://github.com/mo1ein/RenameIt.git
```
then :
```
python3 setup.py
```
and enjoy it :)
### Usage
```
Rit [path] [strftime] [switch]
```
like :
```
Rit /home/moein/pics/photo1.jpg ./photo2.png
```
other ways :
```
Rit /home/moein/lastnight/*jpg Image*.png %c -v -f
```
